Question 2
(a) Solve the equation $x^2 - 2x - 4 = 0$.
Give your answers in the exact form $a \pm \sqrt{b}$, where $a, b \in \mathbb{N}$.
$x =$ $\pm \;\sqrt{\;}$ Not quite. Did you simplify the surd completely?

Spot on! 🎯

Using the quadratic formula for $x^2 - 2x - 4 = 0$ ($A=1, B=-2, C=-4$):

$x = \frac{-(-2) \pm \sqrt{(-2)^2 - 4(1)(-4)}}{2(1)}$

$x = \frac{2 \pm \sqrt{4 + 16}}{2} \;=\; \frac{2 \pm \sqrt{20}}{2}$

Crucial step: Simplify the surd! Find the largest square number that goes into 20.
$\sqrt{20} = \sqrt{4 \times 5} = \sqrt{4}\sqrt{5} = 2\sqrt{5}$

Now substitute it back in:
$x = \frac{2 \pm 2\sqrt{5}}{2}$

Divide both top terms by $2$:
$x = \mathbf{1 \pm \sqrt{5}}$
(So $a = 1, b = 5$).

(b) Given that $(\sqrt{d})^2 = d$, multiply out and simplify $(c + \sqrt{d})^2$.
Enter the three terms below. Order of the 1st and 3rd boxes doesn't matter.
$+$ $\sqrt{d} \quad+$ Check your expansion via FOIL.

Excellent Algebra! 🔥

$(c + \sqrt{d})^2 = (c + \sqrt{d})(c + \sqrt{d})$

Expand using FOIL:
First: $c \times c = c^2$
Outside: $c \times \sqrt{d} = c\sqrt{d}$
Inside: $\sqrt{d} \times c = c\sqrt{d}$
Last: $\sqrt{d} \times \sqrt{d} = d$

Combine the middle terms:
$= c^2 + 2c\sqrt{d} + d$

So the three parts are $\mathbf{c^2}$, $\mathbf{2c}$, and $\mathbf{d}$.

Great Calculation! 🧮

Using the quadratic formula for $2x^2 + 13x - 3 = 0$ ($A=2, B=13, C=-3$):

$x = \frac{-13 \pm \sqrt{13^2 - 4(2)(-3)}}{2(2)}$

$x = \frac{-13 \pm \sqrt{169 + 24}}{4} = \frac{-13 \pm \sqrt{193}}{4}$

$\sqrt{193} \approx 13.8924$

Root 1: $x = \frac{-13 + 13.8924}{4} = \frac{0.8924}{4} = 0.2231 \rightarrow \mathbf{0.223}$
Root 2: $x = \frac{-13 - 13.8924}{4} = \frac{-26.8924}{4} = -6.7231 \rightarrow \mathbf{-6.723}$
